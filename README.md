# Blockbuster Movie Store - Python CLI

This is a Python command-line application that simulates a small movie rental store, inspired by the classic Blockbuster concept. Users can interact with the store through a text-based interface to manage movies and rentals.

## Features

- Add new movies to the store inventory with details like **title**, **release year**, and **rating**.
- View all available movies in the store.
- Rent a movie and keep track of rented movies.
- Return movies and update inventory accordingly.
- Search movies by title or year (optional).
- Exit the application safely.

## Technologies

- Python 3
- Object-Oriented Programming (OOP) concepts: Classes, Objects, Methods
- Dictionaries and Lists for storing movies and rentals
- Command-line interface (CLI)

## How to Use

1. **Clone the repository:**
    ```bash
    git clone <repo-url>
    ```

2. **Run the Python script:**
    ```bash
    python main.py
    ```

3. **Follow the on-screen menu** to:
    - Add new movies
    - View available movies
    - Rent or return movies
    - Exit the application
